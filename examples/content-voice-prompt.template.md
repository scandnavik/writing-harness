# 寫作語感提示詞（範本）

> 這是**範本**，不是某個人的成品。把 `<填你的>` 換成你自己的判準與範例。
> 它的角色：S2 判斷閘「2a 聲音」「2c 在地語感」要拿來比對的那把尺。harness 三站管「不要寫成 AI」，這份檔管「要寫成『你』」。

**當前版本**：v1（你自己往上疊版號）

## 一、口吻核心（用肯定句寫，別寫成禁令清單）

- `<填你的>` 例：用最普通的字把意思講完，不裝熟、不裝專業、不說教
- `<填你的>` 例：不刻意製造口語感（避免硬塞「這傢伙」「白話講」「說真的」）
- `<填你的>` 例：不幫讀者下判斷（不說「值得聽」「極高的參考價值」）

## 二、AI 高頻詞黑名單（這份是通用的，可直接沿用 + 補你常踩的）

寫完搜一遍，命中就換：

- 至關重要 / 不可或缺 / 不容忽視
- 不僅⋯更⋯ / 不只是⋯而是⋯
- 放大影響力 / 賦能 / 賦予⋯的能力
- 具備⋯的能力 / 這取決於 / 值得關注
- 在當今⋯的時代 / 隨著⋯的發展
- 深入探討 / 全面解析 / 一文讀懂
- `<補你自己最常被 AI 塞進來的詞>`

## 三、黃金改寫對照表（最重要，模型只認有範例的規則）

放 3～5 組「AI 會這樣寫 → 我會這樣改」。這是整份檔最有教學力的部分：

| AI 原文 | 你的改法 |
|---------|---------|
| `<貼一句典型 AI 句>` | `<你會怎麼改>` |
| 此技術透過多層架構，實現了從被動工具到主動代理的範式轉移。 | 這個技術用不同的架構，把被動工具變成一種會自己動的工具。 |
| 這對於想掌握未來職涯獲利點的讀者，具備極高的參考價值。 | 如果你想知道未來職涯可以怎麼走，這篇可以多看幾遍。 |
| `<再貼一組你親手改過的>` | `<你的版本>` |

## 四、結構偏好

- 開場結構：`<填你的>` 例：誰在說、說什麼、為什麼現在說，交代時間點與起因
- 時間用自然語言（「昨天的訪談」而非「發布於 2026/3/10」）
- 段落策略：`<填你的>` 例：每個重要論點獨立成段，不合併
- 收尾策略：`<填你的>` 例：不需要收尾段，最後一段若無新資訊就砍

## 五、模型行為補丁（用哪個模型寫就補哪個的坑）

不同模型有不同的頑固慣性，純文字規則容易被跳過，要用「規則＋範例＋工作流檢查」多層攔截：

- `<模型名>` 的慣性：`<填你觀察到的>` 例：很強的「作文要有結尾」慣性，需在多層攔截
- `<模型名>` 的慣性：`<填你觀察到的>` 例：只認真執行有範例的規則，純描述會被跳過

---

**怎麼用**：寫完草稿過 S1 機械閘後，進 S2 時把這份檔調出來，逐條問「這像我寫的，還是像 AI 寫的？」。被真人改稿後，把反覆被改的點回填到第三節對照表——這份檔會越用越準。
