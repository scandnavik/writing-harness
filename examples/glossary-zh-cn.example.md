# 换语境 worked example：简体中文（大陆）

这份文件示范怎么把 `taiwan-writing-glossary.md` 和 `taiwan-style-check.py` 重定向到**简体中文（大陆）**语境。三站方法论（S0/S1/S2）完全不动，只换 glossary 与脚本常数。

把规则分三类处理：**通用保留**、**需翻转**、**需调整**。

---

## 一、通用保留（这些是中文 AI slop 的共性，原样留着）

这几条不分繁简，所有中文都该挡，照抄即可：

- **破折号氾滥**：`——` 过量仍是机器节奏的指纹 → 保留 `EM_DASH` 检查
- **句首噪声框架词**：「其实／老实说／说真的／问题就出在／这正是重点／值得注意的是」零承载填充 → 保留 `CONCLUSION_LEAD_NOISE`
- **「不是 X，是 Y」二元对比模板**：密度上限仍适用 → 保留 `CONTRAST_PATTERN`
- **开场推销词**：「不可不知／再不学就来不及／值得重视」→ 保留 `URGENCY_WORDS`
- **小结格式标记**：`**小结：**` `**Takeaway：**` → 保留 `SUMMARY_LABELS`
- **半角标点夹在中文里**：简中同样用全角标点 → 保留 `check_halfwidth_punct`

## 二、需翻转（繁中专属的判准，对简中要反过来或关掉）

- **§2 词汇对照表（陆 → 台）整段失效**。在简中语境，`数据／项目／软件／信息／视频` 全是正确用词，不该被挡。
  - 做法：把脚本里的 `MAINLAND_WORDS` **清空**（或反向填入你想挡的台湾用词，例如对大陆读者要避免的 `软体／网路／影片／专案`）。
- **引号**：大陆惯例用 `""''`，不用 `「」`。glossary §1.1 的引号那一行要反过来。

## 三、需调整（保留概念，换具体内容）

- **§3.2 英文术语中文化表**：概念保留（科普文该中文化），但对照词换成简中习惯译名（`prompt → 提示词`、`agent → 智能体／代理`，看你的受众）。
- **§2.2 动词 audit 表**：繁中那张表针对台湾口语，简中要换成你自己累积的「书面腔 → 口语」对照。

## 可直接贴的脚本常数（taiwan-style-check.py 顶部）

把 `taiwan-style-check.py` 另存为 `zh-cn-style-check.py`，顶部常数改成：

```python
# 通用保留，照抄
EM_DASH = r"——|──"
SUMMARY_LABELS = [r"\*\*小结[:：]", r"\*\*Takeaway[:：]", r"\*\*重点[:：]"]
URGENCY_WORDS = ["不可不知", "不得不", "你必须", "建议先收藏", "再不学就来不及"]
CONCLUSION_LEAD_NOISE = [
    r"(?:^|[。！？；])\s*(其实|老实说|坦白说|说真的|我记得|不得不说)",
    r"问题就?出在", r"真正(难|值钱|重要|关键)的", r"这正?是重点",
    r"值得(注意|一提)的是",
]
CONTRAST_PATTERN = r"不是.{1,15}，是"

# 需翻转：简中语境下，大陆用词是对的 → 清空。或反向填入你要挡的台湾用词。
MAINLAND_WORDS = []   # 关掉「陆用词」检查
# 若你写给大陆读者、要挡台湾腔，可改成：
# MAINLAND_WORDS = ["软体", "网路", "影片", "专案", "预设", "回报"]

# 半角标点检查（check_halfwidth_punct）原样保留，简中同样用全角
```

其余 `check_*` function 与 `main()` 不用动。跑法不变：

```bash
python zh-cn-style-check.py 你的文章.md
```

港繁、星马华文、纯学术书面中文，欢迎照这个模式做一份 `examples/glossary-<locale>.example.md` 发 PR。方法论（三站）是共享的，你只需要提供那一层「这个语境该挡什么」的判准。
